## electron-vue环境搭建

### window
1. 安装最新版nodejs
2. 全局安装vue-cli
3. 使用**simulatedgreg/electron-vue**模板创建项目即可
```bash
// 安装 vue-cli 和 脚手架样板代码
npm install -g vue-cli
vue init simulatedgreg/electron-vue my-project

// 安装依赖并运行你的程序
cd my-project
yarn // 或者 npm install
yarn run dev # 或者 npm run dev
```
